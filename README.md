# audio_converter
I made this project to improve and test myself in the Python language. As the name suggests, the purpose of the program is to convert audio formats. In addition to that, there is also a separate option for downloading audio from YouTube.
There are 2 main programs. Both are functionally identical. The only difference is that one of them has a transparent background. I made that version as well to test myself.

If you want to convert multiple files at once, create a folder named "input" inside the directory where the exe file is located and place the audio files you want to convert into it. Then, select the "multiple files" option in the program and proceed with your operation.

The exe files are located inside the "dist" folder.
